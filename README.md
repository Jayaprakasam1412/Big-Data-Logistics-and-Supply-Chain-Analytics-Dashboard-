# Big Data Logistics & Supply Chain Analytics

## Project Overview

This project focuses on analyzing logistics and supply chain operations using Big Data technologies. The solution leverages PySpark for large-scale data processing and Power BI for interactive business intelligence dashboards.

The project provides insights into transportation performance, inventory management, supplier effectiveness, and demand forecasting to support data-driven decision-making.

---

## Objectives

* Analyze supply chain and logistics data.
* Monitor transportation costs and delivery performance.
* Perform demand forecasting for inventory planning.
* Evaluate supplier performance and operational efficiency.
* Build interactive dashboards for business intelligence.
